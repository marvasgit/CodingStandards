# Naming types (classes, structures, etc.)

## Use the following formats

[Noun] [Adjective] + [Noun]
Use PascalCase character casing

### Good

```C#
RecursiveFactorialCalculator,
TreeSet,
XmlDocument,
IEnumerable,
Color,
TreeNode,
InvalidTransactionException,
MainForm,
Student,
FileSystem,
BinaryTreeNode,
Constants,
MathUtils,
CheckBox,
Calendar
```

### Bad

```c#
recursiveFactorialCalculator,
recursive_factorial_calculator,
RECURSIVE_FACTORIAL_CALCULATOR,
Move,
FindUsers,
Fast,
ExtremlyFast,
Optimize,
Check,
FastFindInDatabase
```
