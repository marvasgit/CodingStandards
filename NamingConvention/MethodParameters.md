# Naming Method Parameters

- Method parameters names
  - Preferred form: [Noun] or [Adjective] + [Noun]
  - Should be in camelCase
  - Should be meaningful
  - Unit of measure should be obvious

## Good

  ```C#
   firstName,
   report,
   speedKmH,
   usersList,
   fontSizeInPixels,
   font
   ```

## Bad

```C#
p,
p1,
p2,
populate,
LastName,
last_name,
convertImage
```
